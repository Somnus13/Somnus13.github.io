markdown 入门

从最近陆陆续续开始写博客开始，各种笔记，各种转载，然后就发现了有markdown这东西，再加上自己的博客每次发布也是编辑.md文件，所以不要犹豫，学起来！

简介（扒自百科）

Markdown 是一种轻量级标记语言，创始人为約翰·格魯伯(John Gruber)。它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML(或者HTML)文档”。这种语言吸收了很多在电子邮件中已有的纯文本标记的特性。
markdown语法简介明了，功能比纯文本更强，所以很多人用它来写博客。同样也可用于编写说明文档，例如readme.md。简单了解下背景，接下来就是撸语法了。

列表

列表的显示只需要在文字前加上 - 或 * 即可变为无序列表，有序列表则直接在文字前加 1. 2. 3. 符号要和文字之间加上一个字符的空格。
例如：

  • list1
  • list2
  • list3
